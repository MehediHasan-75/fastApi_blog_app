fastapi provides auto docs with Swagger UI and ReDoc UI (/docs and /redoc)

* Python 3.6 with type hints using Pydantic
* based on an open standard:

  * JSON Schema: By default supports returning JSON
  * Open standards: define how you build API
* Code editor auto-complete features
* Security and authentication:

  * HTTP Basic
  * OAuth2 (also with just token)
  * API key in headers, query parameters, cookies
* Testing using pytest
* Supports WebSocket, GraphQL, in-process background tasks, startup and shutdown events
* SQL, NoSQL, GraphQL

**Setup**

```
mkdir project_name
cd project_name
python3 -m venv .venv
source .venv/bin/activate
pip install fastapi
pip install uvicorn
```

**Create main.py**

```python
from fastapi import FastAPI

# creating instance
app = FastAPI()

def index():
    return "hey"
```

**Run**

```
uvicorn main:app --reload
```

* at `127.0.0.1:8000` will show `{"detail" : "Not Found"}`
* So, we need to define path function

**Add decorator for defining path function**

```python
from fastapi import FastAPI

# creating instance
app = FastAPI()

@app.get('/')
def index():
    return "hey"
```

* So, yes, we have built the most straightforward FastAPI api. That's all.
